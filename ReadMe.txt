Krótki poradnik, który należy wykonać przed uruchumieniem programu.

1. Uruchom program Microsoft SQL Server Management Studio (MSSMS)
2. Uruchom kwerendę tworzącą bazę danych - CreateDataBase.sql
3. Przejdź do ktalogu  Projekt Po/lastchance/ i uruchom plik lastchance.csproj
4. W górnym menu Visual Studio wybierz menu Widok, a następnie SQL Sever Object Explorer
5. Utwórz nowe połaczenie z serwerem oraz z wcześniej utworzoną bazą danych
6. W eksploratorze rozwiązań przejdź do pliku App.config 
7. Zlokalizuj linijkę "data source = XYZ;", a następnie wporwadź poprawne dane
8. Uruchom Program
